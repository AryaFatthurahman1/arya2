# PANDUAN PENGGUNAAN HRIS IT/IJK

## Login

1. Buka `http://localhost:8000`
2. Masukkan **nama** atau **email** Anda
3. Masukkan password
4. Klik "Masuk"

## Dashboard

Dashboard menampilkan:
- **Total Karyawan** - Jumlah seluruh karyawan
- **Absensi Hari Ini** - Karyawan yang hadir hari ini
- **Izin Pending** - Pengajuan izin menunggu persetujuan
- **Tugas Pending** - Tugas belum diselesaikan
- **Grafik Absensi** - Statistik 7 hari terakhir
- **Distribusi Status** - Proporsi karyawan tetap/kontrak/percobaan

## Modul Karyawan

### Melihat Daftar Karyawan
Klik menu **Karyawan** di sidebar. Gunakan pencarian untuk menemukan karyawan tertentu.

### Menambah Karyawan
1. Klik **Tambah Karyawan**
2. Isi form data karyawan
3. Upload foto profil (opsional)
4. Klik **Simpan**

### Import dari Excel
1. Klik **Import** di halaman karyawan
2. Upload file Excel (.xlsx)
3. Preview data
4. Klik **Import**

### Export ke Excel
Klik **Export** di halaman karyawan untuk mengunduh data dalam format Excel.

## Modul Absensi

### Absensi Online
1. Klik **Absen Online** di dashboard
2. Sistem akan mencatat waktu masuk Anda

### Import dari Mesin Fingerprint
1. Klik **Import** di halaman absensi
2. Upload file Excel dari mesin fingerprint
3. Klik **Import**

## Modul Pengajuan Izin

### Mengajukan Izin
1. Klik **Ajukan Izin**
2. Pilih jenis izin (sakit/cuti/izin)
3. Isi tanggal dan keterangan
4. Upload bukti dokumen (surat dokter)
5. Klik **Kirim**

### Menyetujui/Menolak Izin (Admin)
1. Klik izin yang pending
2. Lihat detail dan bukti
3. Klik **Setujui** atau **Tolak**

## Modul Penugasan

### Membuat Tugas (Atasan)
1. Klik **Buat Tugas**
2. Isi judul, deskripsi, deadline
3. Pilih karyawan yang ditugaskan
4. Klik **Simpan**

### Update Status Tugas (Karyawan)
1. Buka tugas yang diterima
2. Ubah status (Baru → Diproses → Selesai)
3. Upload bukti penyelesaian
4. Klik **Simpan**

## Modul Penilaian Kinerja

### Membuat Penilaian
1. Klik **Buat Penilaian**
2. Pilih karyawan
3. Isi nilai untuk setiap kriteria
4. Tambahkan catatan
5. Klik **Simpan**

## Modul Penggajian

### Melihat Slip Gaji
1. Klik **Penggajian** di sidebar
2. Klik **Lihat** pada data gaji
3. Klik **Cetak** untuk mencetak slip gaji

## Modul Dokumen

### Upload Dokumen
1. Klik **Tambah Dokumen**
2. Pilih jenis dokumen
3. Upload file
4. Klik **Simpan**

## FAQ

**Q: Lupa password?**
A: Hubungi admin untuk reset password.

**Q: Data tidak muncul?**
A: Refresh halaman atau clear cache browser.

**Q: Import Excel gagal?**
A: Pastikan format file sesuai template yang tersedia.
